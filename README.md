Data sources used in this study

The datasets used in this study are publicly available from the following repositories:

1. Cora, CiteSeer, and PubMed:
https://linqs-data.soe.ucsc.edu/public/lbc/

2. ogbn-arxiv:
https://ogb.stanford.edu/docs/nodeprop/#ogbn-arxiv

3. Chameleon and Squirrel:
https://github.com/graphdml-uiuc-jlu/geom-gcn

4. Penn94 and Pokec:
https://github.com/CUAI/Non-Homophily-Large-Scale
